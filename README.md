# Recuperatorio del Parcial

Instale las dependencias:

```bash
npm install
```

Configure todas las variables de entorno:

```bash
PORT=
DB_NAME=
DB_USER=
DB_PASSWORD=
DB_HOST=
DB_PORT=
DB_DIALECT=
```

Ejecute como desarrollo:

```bash
npm run dev
```
